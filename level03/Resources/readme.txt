В домашней директории лежит файл

-rwsr-sr-x 1 flag03 level03 8627 Mar  5  2016 level03

При запуске он выполняет функцию

system("/usr/bin/env echo Exploit me");

Об этом можно догадаться с помощью команд readelf и strings.
/usr/bin/env вызывается для упрощения эксплойта, чтобы не вызывался
шеловский билт-ин. Поэтому всё, что нам нужно - подменить в системе
команду echo. В системе немного мест, куда можно писать файлы, одно из
них /tmp. Туда я положил файл echo с правами на выполнение, внутри
которого вызывается /bin/getflag. После чего заменил содержимое PATH в
шеле на /tmp. В итоге выполнение ./level03 дало нужный флаг.
