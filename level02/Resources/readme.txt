В домашней директории лежит файл level02.pcap. Скачиваем через scp,
открываем в wireshark. Внутри содержится трафик, который передавался
между двумя компами по TCP протоколу. В какой-то момент один комп
спрашивает "Password:", а второй отвечает следующей
последовательностью:

[f][t][_][w][a][n][d][r][7f][7f][7f][N][D][R][e][l][7f][L][0][L]

На что первый отвечает "Log in incorrect"

7f означает DEL (backspace). Итоговый пароль "ft_waNDReL0L"
